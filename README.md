#  Solanyte 


A mobile wallet tracking application for iOS.

### Usage
- Gives extended description of each coin (powered by Coingecko)
- Tracks token accounts of wallet
- Stores multiple wallet addresses
- Stores balance of current wallet

### Powered by:
- Solana
- Coingecko
- Solscan
